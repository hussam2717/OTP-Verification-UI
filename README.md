# OTP-Verification-UI
A clean and modern OTP (One-Time Password) input interface built using HTML, CSS, and JavaScript. This project simulates a verification code input screen commonly used in authentication flows.

Live Demo - 

## 🚀 Features
- 6-digit OTP input fields
- Auto focus shift between inputs
- Backspace navigation support
- Clean and minimal UI design
- Responsive layout
- "Resend Code" option UI
